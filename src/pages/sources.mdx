# Page orientation :

Haute Autorité de Santé (HAS) - Sur les outils d'observation dans les carnets de santé pour le dépistage des troubles du développement : Haute Autorité de Santé (HAS) 

Maison Départementale des Personnes Handicapées (MDPH) - Pour les informations sur les démarches et les évaluations : Site officiel des MDPH 

Centre de Ressources Autisme (CRA) - Informations sur les diagnostics et les évaluations : Réseau National des CRA 

Approches éducatives comme ABA, TEACCH, ESDM, PECS - Plusieurs recherches et articles spécialisés sur ces méthodes, par exemple sur le site de l'INSERM ou de l'HAS : INSERM - Troubles du spectre de l’autisme 

Structures d'accompagnement comme les SESSAD, PCO, SAVS, et SAMSAH - Informations générales sur ces dispositifs via les portails publics comme celui de l’Assurance Maladie : Ameli - Autisme 

Institut Médico-Éducatif (IME) et autres établissements spécialisés - Portail gouvernemental pour les dispositifs médico-sociaux : Service Public - Handicap 

Modèle SACCADE et Langage SACCADE Conceptuel (LSC) - Informations spécifiques sur ce modèle développées par les auteurs ou sur les sites spécialisés en autisme : SACCADE 

Financement des structures et accompagnements - CNSA pour des précisions sur le financement : CNSA - Politiques publiques et financements
